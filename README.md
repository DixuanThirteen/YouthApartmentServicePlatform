青年公寓服务平台
=========================
* 本项目为动态网站设计实训大作业作品
* 本项目为半成品
* ***本项目目前使用 JDK 17 、 Spring Boot 、 Mybatis 、 Vue.js等技术构成***
* **本项目作业提交时间为<mark> 2025.12.24 </mark>**

-----------------
本项目的大致结构为:
    
    src              ← java
    ├── controller   ← 控制层（C）
    ├── service      ← 业务层
    ├── entity       ← 实体层（Model 的一部分）
    ├── mapper       ← 数据访问层（DAO）
    └── dto          ← 数据传输对象
    vue_yasp         ← vue前端（view层)
